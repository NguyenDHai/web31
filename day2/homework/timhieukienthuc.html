Bao gồm 3 thành phần: block, element và modifier

Quy tắc đặt tên

Block

Tên block có thể bao gồm các chữ cái Latinh, chữ số và dấu gạch ngang.

Tạo CSS class: thêm một tiền tố vào phía trước.

Element

Tên element có thể bao gồm các chũ Latinh, chữ số, dấu gạch ngang và dấu gạch dưới.

Tạo class CSS: tên block cộng với hai dấu gạch dưới.

Modifier

Tên của modifier có thể bao gồm các chữ Latinh, chữ số, dấu gạch ngang và dấu gạch dưới.

Tạo class CSS: tên block hoặc element cộng với hai dấu gạch ngang, cộng với modifier.

Dấu cách trong các modifier dài (chứa 2 tiếng trở lên) được thay thế bằng dấu gạch ngang

Box Model là một kỹ thuật cơ bản nhất trong CSS Layout và được sử dụng để bạn mô tả về khoảng cách mà mỗi phần tử trên website được sở hữu, hay nói cách khác là kỹ thuật tinh chỉnh khoảng cách hiển thị cho mỗi phần tử trên website.

Kỹ thuật Box Model trong CSS bao gồm 4 phần quan trọng đó là:
  • Margin: Khoảng cách tính từ bên ngoài của phần tử.
  • Border: Đường viền cảu phần tử.
  • Padding: Khoảng cách tính từ bên trong của phần tử.
  • Content: Nội dung trong phần tử.
